# Drone Hopper - Professional Website

![Drone Hopper Logo](icons/logo.png)

**Drone Hopper S.L.U.** is a Madrid-based aeronautical engineering company specializing in the design, manufacturing, and commercialization of **high-payload unmanned aerial systems (UAS/drones)** for industrial and professional applications.

This repository contains the source code for the official professional website of **Drone Hopper**, showcasing our innovative drone solutions, engineering expertise, and commitment to safety, efficiency, and sustainability.

## About Drone Hopper

Drone Hopper develops heavy-duty commercial drones with advanced features:
- High payload capacity (up to hundreds of kg)
- Long endurance and autonomy
- Patented technologies (e.g., liquid misting/nebulization systems for firefighting)
- Modular platforms for multiple missions: firefighting, agriculture (precision spraying), logistics, infrastructure inspection, surveillance, and more.

Our flagship models and hybrid/thermal-powered platforms set us apart in the UAV industry, targeting sectors where standard consumer drones fall short.
